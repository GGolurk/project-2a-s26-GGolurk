# project-2a-s26-GGolurk

## By Nathan Merk

This project is here to share the best wave breaker locations in Splatoon 3.
Wave breaker is very map dependent, and certain wave locations may benefit from
special power up while others may not, so it's important to know where to throw
it, how much SPU is needed, and whether the map you're on is a good one for it.
